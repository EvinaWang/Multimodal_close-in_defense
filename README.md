# 1 系统整体架构

## 1.1 传感器配置说明

系统采用以下多模态传感器组合，安装于二轴稳定云台之上：

- 普通RGB摄像头：负责日间高分辨率纹理采集与目标分类。
- 红外热成像摄像头：负责夜间、雾霾等低能见度环境下的目标探测。
- 77GHz相控阵毫米波雷达：提供精确的径向速度、距离及空间不确定性信息。
- 激光发射器：作为硬杀伤或软压制载荷，配合火控系统进行精确打击。
- 二轴云台：提供方位与俯仰自由度，支持高速目标跟踪与激光指向。

## 1.2 系统工作流程

系统遵循"感知 → 融合 → 决策 → 执行"的闭环逻辑：

1. 数据采集：多传感器同步采集，经Reinhard增强及预处理。
2. 特征提取：视觉分支通过LSS转换至BEV空间，雷达分支投影至BEV空间。
3. 时空融合：在BEV空间进行门控注意力融合，利用RaUF不确定性调制权重。
4. 检测跟踪：CenterPoint输出3D框，结合时序建模进行稳定跟踪。
5. 火控解算：威胁评估、路径优化、延迟补偿及激光发射。

## 1.3 模块功能划分



| 模块层级   | 核心功能           | 关键技术                               |
| :--------- | :----------------- | :------------------------------------- |
| **感知层** | 异构数据特征提取   | YOLO改进、LSS深度估计、RaUF不确定性    |
| **融合层** | 多源信息对齐与加权 | 联合标定、PTP同步、门控注意力机制      |
| **决策层** | 威胁排序与打击规划 | 威胁评分模型、模拟退火路径优化         |
| **执行层** | 精确指向与发射控制 | 坐标转换、飞行时间补偿、云台运动学约束 |
